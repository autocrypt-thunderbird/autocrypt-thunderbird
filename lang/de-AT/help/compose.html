OpenPGP-Hilfe

Verwenden von OpenPGP beim Verfassen von Nachrichten

Menü "OpenPGP" im Fenster "Verfassen"
Nachricht unterschreiben
Unterschreiben von Nachrichten aktivieren / deaktivieren. Der Benutzer wird benachrichtigt, falls das Unterschreiben fehlschlägt.
Nachricht verschlüsseln

Verschlüsselung an alle Adressaten aktivieren / deaktivieren. Der Benutzer wird benachrichtigt, falls die Verschlüsselung fehlschlägt.

Falls Auswahl anzeigen, wenn notwendig im Menü OpenPGP -> Einstellungen -> Schlüsselauswahl... eingestellt ist, dann wird eine Liste mit Schlüsseln erscheinen, wenn Sie keinen öffentlichen Schlüssel für einen oder mehrere Empfänger der Nachrichten besitzen.

Falls Auswahlfenster nie anzeigen im Menü OpenPGP -> Einstellungen -> Schlüsselauswahl... eingestellt ist, dann wird die Nachricht an die Empfänger unverschlüsselt gesendet, von denen Sie keinen öffentlichen Schlüssel besitzen.

PGP/MIME verwenden
Verwenden von PGP/MIME für diese Nachricht ermöglichen / nicht ermöglichen.

Wenn Sie wissen, dass die Empfänger das PGP/MIME-Format verwenden können, dann sollten Sie dies benutzen.

Diese Möglichkeit hängt von den Einstellungen unter OpenPGP -> Einstellungen -> PGP/MIME... ab. Hier muss PGP/MIME verwenden, wenn möglich oder Immer PGP/MIME verwenden aktiviert sein.

Einstellungen (Untermenü)
  • Unterschrift/Verschlüsselung...: Abkürzung zu den Konten-Einstellungen -> OpenPGP-Sicherheit.
  • Senden...: Abkürzung zu Einstellungen -> Senden.
  • Schlüsselwauswahl...: Abkürzung zu Einstellungen -> Schlüsselauswahl.
  • PGP/MIME...: Abkürzung zu Einstellungen -> PGP/MIME.
Verschlüsselung rückgängig machen

Wenn es einen Fehler beim Senden der Nachricht gibt, also z.B. der POP-Server nicht erreichbar ist, dann wird OpenPGP dies nicht wissen und die verschlüsselte Nachricht wird auch weiterhin im Fenster "Verfassen" angezeigt. Dieser Menüpunkt nimmt die Verschlüsselung / das Unterschreiben zurück und das Fenster "Verfassen" kehrt zum Originaltext zurück.

Als vorläufige Maßnahme kann über diesen Menüpunkt auch eine verschlüsselte Nachricht, die man zitiert und auf die man antwortet, entschlüsselt angezeigt werden. Enigmail sollte zitierte Nachrichten automatisch entschlüsseln, was bisher jedoch fehlschlägt. Über diesen Menüeintrag kann Enigmail zur Entschlüsselung gezwungen werden.

Öffentlichen Schlüssel einfügen
fügt an der Cursorposition den öffentlichen Schlüsselblock im ASCII-Format ein. Sie werden gefragt, von welchen E-Mail-Adressen die Schlüssel eingefügt werden sollen. Schlüssel, die auf diese Weise eingefügt werden, werden automatisch vom Enigmail des Empfängers entdeckt. Es ist weiterhin möglich, die Nachricht auch nach Einfügen eines Schlüsselblocks zu unterschreiben / verschlüsseln. Bitte nicht mehr als einen Schlüsselblock separat einfügen; geben Sie einfach beim ersten Einfügen alle E-Mail-Adressen an (durch Kommas oder Leerzeichen getrennt), die eingefügt werden sollen.
Passphrase aus Cache löschen
Löscht die gespeicherte Passphrase. Nützlich, falls Sie mehrere Passphrasen verwenden.
